# portfolio-hub

`portfolio-hub` is the main personal portfolio website for Gidion Ondari.

## Purpose

This project presents:
- Professional identity and technical focus.
- Active showcase builds and coming-soon projects.
- A clear progression from learning/school projects toward production-grade development.

## Stack

- React
- Vite
- Plain CSS

## Current Scope

This is a polished single-page homepage with:
- Navbar
- Hero
- About
- Skills
- Featured Projects
- Secondary Projects
- Experience
- Build Process
- Contact
- Footer

## Local Development

```bash
npm install
npm run dev
```

## Build Check

```bash
npm run build
```

## Deployment Notes (Vercel)

To deploy `portfolio-hub` on Vercel, configure:
- **Root Directory:** `portfolio-hub`
- **Build Command:** `npm run build`
- **Output Directory:** `dist`

Project links currently use placeholders (`#`) until deployments and repository links are finalized.
