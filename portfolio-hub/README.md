# portfolio-hub

`portfolio-hub` is the main personal portfolio website for Gidion Ondari.

## Purpose

This project presents:
- Professional identity and technical focus.
- Active showcase builds and coming-soon projects.
- A clear progression from learning/school projects toward production-grade development.

## Visual System

This app now uses the shared portfolio brand palette:
- Black background foundation
- Green for live/progress states
- Red for controlled CTA/warning accents
- White for readability and contrast

## Stack

- React
- Vite
- Plain CSS
- Frontend-only prototype implementation

## Local Development

```bash
npm install
npm run dev
```

## Build Check

```bash
npm run build
```
