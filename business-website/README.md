# Business Website (PureGlow Naturals)

## Purpose
A premium small-business skincare website prototype for PureGlow Naturals focused on product clarity, trust-building communication, and direct WhatsApp inquiry conversion.

## Tech Stack
- React
- Vite
- Plain CSS

## Visual System
Unified portfolio brand palette:
- Black base/background for premium depth
- Green for freshness, natural identity, and positive status cues
- Red for controlled CTA/action accents
- White/muted text for readability

## Features
- Hero with PureGlow Naturals brand intro and WhatsApp CTA
- Trust and value cards
- 6-product showcase with mock pricing and WhatsApp inquiry actions
- Service offers and customer trust section
- 4-step ordering flow
- Demo-labeled testimonials/social proof
- Business metrics trust layer
- Contact CTA with WhatsApp and email actions

## Disclaimer
This is a frontend-only portfolio prototype. It is not a live ecommerce system and does not include checkout, payment processing, backend order management, or live inventory.

## Run Locally
```bash
npm install
npm run dev
npm run build
npm run preview
```

## Portfolio Relevance
This project demonstrates small-business web design execution, trust-oriented product presentation, and practical conversion UX aligned with the portfolio’s unified black/green/red/white brand identity.
