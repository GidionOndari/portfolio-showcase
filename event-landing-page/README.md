# Event Landing Page — Finalist Costume Dinner (Class of 2026 Kenya)

## Purpose
This project is a premium sponsor-facing event campaign prototype for Gidion Ondari's portfolio. It showcases event storytelling, conversion-focused landing page structure, sponsor value communication, and responsive frontend presentation.

## Tech Stack
- React
- Vite
- Plain CSS
- Static mock data only (no backend)

## Features
- Premium hero with campaign positioning and conversion CTAs
- Event value cards for celebration, fashion, media, and sponsor visibility
- Program flow table with stage labels
- Sponsor visibility section with activation checklist
- Venue layout concept section (non-confirmed conceptual layout)
- Audience and impact framing
- Visual moments gallery placeholders
- Closing CTA cards and organizer contacts
- Gamified UI blocks: experience score, impact meter, readiness level, moment badges

## Event Concept Disclaimer
This is a frontend portfolio prototype and event campaign concept. It is not a live ticketing/payment platform and does not represent confirmed sponsor or venue agreements.

## Run Locally
```bash
npm install
npm run dev
npm run build
npm run preview
```

## Portfolio Relevance
The page demonstrates campaign/product thinking, sponsor-facing communication, event activation strategy translation, and UI execution quality suitable for recruiters, partners, and clients.
